# angular8-simple-example

- Install angular 

             npm install -g @angular/cli

- Create new angular 8 project 
              
             ng new angular-example
             
 - Start Server 
 
            cd angular-example  
            
            
            ng serve
            
- Open Browser 

            http://localhost:4200/
